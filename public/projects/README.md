# Project cover screenshots

Real screenshots of each live site, named `<slug>.png` where the slug matches
the `image` field of each entry in `lib/content.ts`
(e.g. `med-tohokukosai.png`).

- Any file present is shown as that card's cover.
- Any file missing (some sites block automated capture behind a firewall) falls
  back to a generated mockup automatically, so the site never looks broken.

## Status

Most projects have a real site screenshot. To refresh one, capture it
(landscape, ~1440×900) and overwrite the file here.
